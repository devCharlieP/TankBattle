# TankBattle

탱크를 움직이고 탄환을 발사해 건너편 상대 탱크를 부수는 게임입니다.
OpenGL, GLEW, GLFW3를 이용해 클라이언트를 Boost asio를 이용해 동기식으로 서버를 구현하였습니다.
